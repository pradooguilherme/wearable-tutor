# Projeto Computação Pervasiva e Ubíqua #

## Wearable Tutor ##

Passo a Passo de Inicialização do ESP32

- Conectar ESP32 no RaspberrryPI, com as ligações do LED da aula 3 (Sensores). Usando o Thonny, trocar o interpretador em "Executar" -> "Configurar interpretador", e alterar para MicroPythonESP32, e selecionar o dispositivo
e rodar o programa main.py para descobrir o IP do ESP32.

- Após descobrir o IP do ESP32, parar o código do mesmo e inserir o código no ESP32, de forma que rode automaticamente.

- O restante só rodar na pasta .venv.
